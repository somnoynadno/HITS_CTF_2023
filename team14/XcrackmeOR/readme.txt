Сей crackme написан на C++, без использования обфускаторов/antiVM/antiDebug


Динамическая генерация ключа, т.е. ключ генерируется при вводе имени, если ввести правильный ключ, высветится флаг
Флаг статичен, однако через просмотр констант в дебаггере полностью не отсвечивается, т.к. составляется посимвольно
При вводе неправильных символов имени, приложение тебя буллит


Принцип работы такой, что каждый символ имени xor'ится с 1010101


Для решения можно применить дебаггер, отследив то, что происходит с памятью 
Либо брут, введя имя в один символ и перебрав все символы таблицы ascii
